# aniruddhzaveri.github.io

Personal portfolio of **Aniruddh Zaveri** — Software Engineer & Agile Coach at TUM (Applied Education Technologies), building [TUMApply](https://aet.cit.tum.de/projects/edtech/tum-apply/).

Built with React 19, TypeScript, Tailwind CSS v4, shadcn/ui and Motion. Bilingual (EN/DE), dark teal×indigo theme, scroll-driven experience timeline.

## Development

```bash
npm install
npm run dev        # dev server
npm test           # vitest
npm run build      # production build
```

## Deployment

Pushes to `main` deploy automatically to GitHub Pages via `.github/workflows/deploy.yml`. The base path adapts to the repo name (user site vs. project page).
